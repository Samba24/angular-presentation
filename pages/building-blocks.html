3 - Les éléments constitutifs d'Angular

3.1 - NgModules

Les modules définissent un contexte d'exécution pour des composants spécifiques. A la creation d'un projet Angular avec la CLI un module de base (root module) est automatiquement créé et regroupe tous les elements que l'on declare dans notre application tel que les composants, les services ou les modules tiers. Les NgModules utilise le décorateur @NgModule et disposent de quatre arrays utilisés pour les inclusions que sont :

  • Declarations

    Regroupe tous les elements susceptibles d'être utilisé dans nos templates tel que les composants, pipes et directives et qui sont uniquement visibles dans le module actuel (specific scope).

  • Providers

    On retrouve tous les services enregistrés dans notre module dans cet array.

  • Imports

    Regroupe tous les autres modules qui sont importés dans le module courant.

  • Bootstrap

    Permet de définir le composant de base (root component) lors du processus de bootstraping.

3.2 - Composants

Les composants sont les pièces maîtresses d'une application Angular. Ils sont en charge de contrôler les templates en leur fournissant leurs données ainsi qu'en gérant toute la logique derrière ces derniers.
Les composants utilisent le décorateur @Component et sont associés à un template a leur creation.
On peut facilement créer un composant en utilisant la commande ng generate component nom_du_composant ou son raccourci ng g c nom_du_composant. On obtient ainsi les fichiers suivants dans un dossier portant le nom de notre composant sous le répertoire app

myComponent

myComponent.component.ts

Correspond a notre composant en tant que tel, toute la logique permettant de piloter la vue se trouve dans ce fichier.

myComponent.component.spec.ts

Permet d'écrire des tests sur notre composant a l'aide de Jasmine (optionnelle).

myComponent.component.css

Permet de définir des styles CSS sur la vue associé a notre composant.

myComponent.component.html

Représente la vue du composant, toute la partie presentation se trouve dans ce fichier.
